# KeyGen
KeyGen é um gerador de key JSK para Play Store

Ele automatiza a criação de certificados JSK

#### Update do termux
``` shell script
apt update -y;
apt upgrade -y;
```
#### install 

``` shell script 
pkg install wget -y;
wget https://github.com/MTplusWebSystem/KeyGen/blob/main/keygen.sh;
chmod 777 keygen.sh;
 ./keygen.sh;

```
